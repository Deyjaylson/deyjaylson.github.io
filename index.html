Imagem 1 - Protocolos Web

5) Leia o Enunciado: "Tim Berners-Lee é considerado o pai da Internet: ele, simplesmente, criou o HTTP (Hyper Text Transfer Protocol)"

Assinale abaixo o que é o HTTP:

A) HTTP é um protocolo de transporte confiável baseado em datagramas, que garante a entrega sequencial e sem perdas de pacotes através da camada de transporte TCP/IP.

B) HTTP é um protocolo de aplicação que funciona em modo stateless, operando essencialmente como um protocolo de solicitação-resposta entre clientes e servidores, sem manutenção de estado entre conexões.

C) HTTP foi originalmente concebido para transmissão de arquivos binários de forma eficiente, incorporando mecanismos nativos de compressão de dados e criptografia de ponta a ponta desde sua versão inicial.

D) HTTP é um protocolo de rede de camada de enlace que define os endereçamentos físicos de dispositivos conectados à Internet e gerencia a troca de frames entre roteadores.

6) Leia o Enunciado: "Tim Berners-Lee é considerado o pai da Internet: ele, simplesmente, criou o HTML(Hyper Text markup Language)"

Assinale abaixo o que é o HTML:

A) HTML é uma linguagem de programação compilada, que permite a criação de aplicativos web complexos com lógica de execução no lado do servidor.

B) HTML é uma linguagem de marcação de hipertexto, utilizada para estruturar e apresentar conteúdo em páginas web, sendo interpretada pelo navegador sem necessidade de compilação.

C) HTML é um protocolo de rede que define como dados são transmitidos entre clientes e servidores, garantindo comunicação confiável sobre a Internet.

D) HTML é um sistema de gerenciamento de banco de dados relacional, responsável pelo armazenamento e recuperação de informações em websites dinâmicos.

7) Leia o Enunciado: "Tim Berners-Lee é considerado o pai da Internet: ele, simplesmente, criou o NEXUS"

Assinale abaixo o que é o NEXUS:

A) NEXUS foi o primeiro navegador web, criado em 1990, e ainda hoje é amplamente utilizado em navegadores modernos como padrão corporativo.

B) NEXUS é uma ferramenta de nuvem, usada para armazenar, versionar e distribuir bibliotecas, pacotes e artefatos em projetos de desenvolvimento.

C) NEXUS é um protocolo de rede responsável pelo roteamento de pacotes IP em ambientes distribuídos, garantindo entrega confiável e balanceamento de carga.

D) NEXUS é um sistema de compressão de arquivos multimídia, projetado para otimizar a transmissão de vídeos e áudios em redes de alta velocidade.
